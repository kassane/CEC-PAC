# CEC-PAC

 This repository holds the Peripheral Access Crates (PAC) for [Microchip CEC Series Device Support](https://packs.download.microchip.com/#collapse-Microchip-CEC-DFP-pdsc) v1.5.142.


## Devices

- CEC1702
- CEC1712H_B2_SX
- CEC1712H_N2_SX
- CEC1712H_S2_SX
- CEC1734_S0_2HW
- CEC1734_S0_2ZW
- CEC1734_S0_SX
- CEC1736_S0_2HW
- CEC1736_S0_2ZW
- CEC1736_S0_SX
